# VibeKTV

一个可直接打开的 KTV 点歌系统原型。

## 已完成

- 点歌台式界面
- 歌曲分类筛选
- 收藏分类和最近播放分类
- 歌名 / 歌手 / 标签搜索
- 已点队列
- 队列置顶、立即播放、移除
- 播放、暂停、切歌、重唱
- 歌词同步滚动
- LRC 歌词文件导入
- 伴奏 / 原唱状态切换
- 音乐音量、麦克风音量、混响控制
- 音量设置本地记忆
- 收藏和最近播放本地记忆
- 本地音频导入
- 演唱全屏模式
- 模拟评分
- 离线示例伴奏：内置示例歌不依赖网络 MP3

## 打开方式

直接用浏览器打开项目根目录里的 `index.html`：

```text
index.html
```

也可以在项目根目录启动本地服务：

```text
python -m http.server 4173 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:4173/
```

## 使用说明

1. 在左侧选择分类，或在顶部搜索歌曲。
2. 点击「点歌」加入已点队列。
3. 已点队列支持播放、置顶、移除。
4. 中间舞台区会显示当前歌曲和滚动歌词。
5. 示例歌曲会播放浏览器合成的简单伴奏。
6. 点击「导入本地歌曲」可以选择本机音频或视频文件播放。
7. 播放当前歌曲后，点击「导入 LRC 歌词」可以替换当前歌词。
8. 麦克风监听需要浏览器允许麦克风权限；如果本地文件模式限制权限，建议用本地服务或升级 Electron 桌面版。

## 快捷键

- Space：播放 / 暂停
- N：切歌
- R：重唱
- F：演唱全屏 / 退出
